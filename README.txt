PAI.DUY.NA LOCAL ADMIN

1. เปิด PAI_DUY_NA_Admin.html ด้วยเว็บเบราว์เซอร์
2. กรอกข่าวสถานะและกด Export status.json
3. นำ status.json ไปแทนไฟล์ paiduyna/data/status.json
4. Commit และ Push เฉพาะโฟลเดอร์ paiduyna ขึ้น GitHub Pages

ห้ามอัปโหลดโฟลเดอร์ admin-local ขึ้นเว็บไซต์สาธารณะ
สิทธิ์เผยแพร่ข้อมูลจริงควรให้เฉพาะบัญชี GitHub ของผู้ดูแล Repository เท่านั้น
